This project is a small Python utility for collecting metadata and downloading
YouTube videos in batches. It is designed for simple keyword-based workflows
and quick local automation.

The repository includes scripts for metadata collection, category management,
and bulk downloading, making it useful for lightweight personal archiving or
dataset preparation.

Project link: [GitHub Repository](https://github.com/Cryo3978/Youtube-Downloader)
